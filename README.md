# DeepSeek v4 for Visual Studio

这是一个用于 Visual Studio 的扩展，将 DeepSeek Chat 聊天窗口集成到了 IDE 中，让开发者可以一边查看代码一边与 DeepSeek AI 无缝交互。

## 特性 (Features)

- **DeepSeek Chat 工具窗口**：在 Visual Studio 内置并可自由停靠的聊天工具窗口。
- **联网搜索**：支持联网数据搜集，以获取有关开发和错误的最新信息。
- **聊天记录存档**：自动保存您的所有对话记录，方便下次继续交流。
- **可定制选项**：内置设置页面，可在 Visual Studio 的“工具”->“选项”中快速配置 API 密钥和其他首选项。
- **快速入口**：
  - **菜单栏**：通过 `视图 (View)` -> `其他窗口 (Other Windows)` -> `DeepSeek Chat` 打开。
  - **工具栏**：标准工具栏已集成快捷按钮，一键调出 AI 助手。

## 安装 (Installation)

1. 通过 Release 页面下载最新的 `.vsix` 扩展包。
2. 双击安装并应用到您的 Visual Studio 中。
3. 重启 Visual Studio 即可使用。

## 使用说明 (Usage)

安装完成后，你可以通过上述的“视图”菜单或者工具栏按钮来打开侧边栏。在窗口中配置好相关的 API 密钥后，即可开始与 DeepSeek 互动（例如解释代码、重构建议、Bug 修复等）。

# DeepSeek v4 for Visual Studio (English)

This is an extension for Visual Studio that integrates the DeepSeek Chat window into the IDE, allowing developers to interact seamlessly with DeepSeek AI while viewing their code.

## Features

- **DeepSeek Chat Tool Window**: A built-in and dockable chat tool window in Visual Studio.
- **Web Search Support**: Integrated web search capabilities to fetch up-to-date information for development and coding bugs.
- **Chat History Persistence**: Chat contexts are automatically saved locally across sessions.
- **Configuration Options**: Easily configure your API keys and model preferences via `Tools` -> `Options` in Visual Studio.
- **Quick Access**:
  - **Menu Bar**: Open via `View` -> `Other Windows` -> `DeepSeek Chat`.
  - **Toolbar**: A quick-access button is integrated into the standard toolbar to summon the AI assistant with one click.

## Installation

1. Download the latest `.vsix` extension package from the Release page.
2. Double-click to install and apply it to your Visual Studio.
3. Restart Visual Studio to use the extension.

## Usage

After installation, you can open the sidebar using the "View" menu or the toolbar button mentioned above. Once you configure the relevant API keys in the window, you can start interacting with DeepSeek (e.g., code explanation, refactoring suggestions, bug fixing, etc.).
